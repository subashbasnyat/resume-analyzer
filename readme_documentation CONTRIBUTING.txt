Steps for creating good issues or pull requests.
  1. before commiting, check "git checkout -b <branch name>
                              git status
                              git diff
  2. To write changes, write "git add <filepath> "
  3. git commit -m <messages regarding changes>. E.g. git commit -m "Implement if else conditions for xyz document"
  4. git branch
  5. git push origin <branch name>
